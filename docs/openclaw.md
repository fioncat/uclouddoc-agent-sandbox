# OpenClaw 快速接入

利用Sandbox技术，我们可以在几十秒内快速启动一个OpenClaw服务并直接使用。

为了方便使用，我们为您提供了一个简易的网站，可以让您快速利用Sandbox部署OpenClaw服务，无需再使用SDK/CLI：[OpenClaw](https://openclaw.sandbox.ucloudai.com/)。

进入网页之后，在下面的红框输入框内输入您的ModelVerse Key，即可启动一个OpenClaw服务（Key可以在[控制台](https://console.ucloud.cn/modelverse/experience/api-keys)获取）:

![openclaw-01](/images/openclaw-01.png)

等待初始化完成之后，可以直接进入OpenClaw管理页面：

![openclaw-02](/images/openclaw-02.png)
