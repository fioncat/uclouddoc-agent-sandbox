# 切换地域

我们的Sandbox支持多地域切换，您可以根据您的实际情况选择对应的地域。目前支持的地域列表：

| 地域    | 中文描述         | 域名                         |
| ------- | ---------------- | ---------------------------- |
| cn-wlcb | 华北（乌兰察布） | cn-wlcb.sandbox.ucloudai.com |
| us-ca   | 美国（洛杉矶）   | us-ca.sandbox.ucloudai.com   |

在默认情况下，我们将使用 `cn-wlcb` 地域。下面介绍如何切换到别的地域。

> ⚠️注意：不同地域的沙箱和模板数据是独立且不互通的，您的自定义模板在不同地域都需要单独构建一份。

## SDK 切换地域

您可以通过 `AGENTBOX_REGION` 环境变量来直接切换地域。（切换地域要求 SDK 版本大于等于`1.2.0`）

```bash
export AGENTBOX_REGION=xxx
```

例如，假如您希望将沙箱部署在北美，则设置这个环境变量为`us-ca`。

## CLI 切换地域

直接执行下面的命令，可以实现一键切换地域。（切换地域要求 CLI 版本大于等于`1.0.3`）

```bash
ucloud-sandbox-cli region
```

命令行会提示您选择地域，选择之后将会全局生效。
